# cse425-beta-vae-music-clustering

Unsupervised music clustering using a Beta-Hybrid VAE (audio + optional lyrics).
Author: Saidul Karim (22301497)

## Project Structure
- notebooks/ : Colab notebooks
- src/       : Python modules (VAE, dataset, clustering, evaluation)
- results/   : Outputs (csv, plots)
- figures/   : Paper figures (png)
